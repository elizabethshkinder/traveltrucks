# TravelTrucks

Фронтенд-застосунок для сервісу оренди кемперів, створений з використанням Next.js та TypeScript.

## Опис
TravelTrucks — це вебзастосунок для перегляду, фільтрації та бронювання кемперів. Він містить головну сторінку, каталог з фільтрами та пагінацією, а також сторінку деталей з відгуками і формою бронювання.

## Функціонал
- Головна сторінка з hero-блоком
- Каталог кемперів
- Фільтрація за локацією, типом кузова, двигуном і трансмісією
- Пагінація Load More
- Сторінка деталей кемпера
- Галерея зображень
- Відгуки з рейтингом
- Форма бронювання
- Нотифікація після успішного відправлення
- Інтеграція з бекендом

## Технології
- Next.js (App Router)
- TypeScript
- TanStack Query
- Axios
- Formik
- Yup
- CSS Modules
- React Icons
- React Hot Toast
