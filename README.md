# TravelTrucks

Frontend application for a camper rental service built with Next.js and TypeScript.

## Overview
TravelTrucks is a web application for browsing, filtering, and booking camper vans. It includes a home page, a camper catalog with filters and pagination, and a detailed camper page with reviews and a booking form.

## Features
- Home page with hero section and CTA
- Camper catalog page
- Filtering by location, camper form, engine type, transmission
- Pagination with Load More
- Camper details page
- Image gallery
- Reviews with 5-star rating display
- Booking form with validation
- Success notification after form submission
- Backend integration via API

## Tech Stack
- Next.js (App Router)
- TypeScript
- TanStack Query
- Axios
- Formik
- Yup
- CSS Modules
- React Icons
- React Hot Toast
